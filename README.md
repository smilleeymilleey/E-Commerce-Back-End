## About The Project





 I wanted to create an easy way to create, read, update and delete product information from a database for an e-commerce site!  

Here's why its awesome:
* You can create product, categories, and product tag information
* You can read product, categories, and product tag information
* You can update product, categories, and product tag information
* You can delete product, categories, and product tag information

Thanks for checking it out!

A list of commonly used resources that I find helpful are listed in the acknowledgements.

### Built With

* [Javascript]
* [Express,js]
* [Node.js]
* [mySQL]
